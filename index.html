Handwriting Recognition of Numbers using CNN

The project is about implementing a Handwritten Character Recognition model using TensorFlow Convolution Neural Network model. MNIST database is used to train and predict model and check the accuracy of the model. In CNN, two layers of convolution is provided. Then the maxpool function is applied to the result to obtain weighted images. On applying ReLU, images were normalized. Finally the training prediction accuracy using ADAM Optimizer is checked.

Spectral Clustering - Image Segmentation & Object Recognition

The project implements partitioning an image into several disjoint subsets such that each subset corresponds to a meaningful part of the image. The project makes use of the spectrum (eigenvalues) of the similarity matrix of the data to perform dimensionality reduction before clustering in fewer dimensions. The similarity matrix is provided as an input and consists of a quantitative assessment of the relative similarity of each pair of points in the dataset.
